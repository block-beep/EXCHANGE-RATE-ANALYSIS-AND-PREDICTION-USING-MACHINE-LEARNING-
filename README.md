💱 Exchange Rate Analysis and Prediction using Machine Learning

📌 Project Overview

This project focuses on analyzing and predicting exchange rates of currencies like USD, EUR, GBP, and CNY using Machine Learning techniques. It uses historical data and forecasts future trends for better financial insights.

🎯 Objective

* Analyze historical exchange rate trends
* Build a prediction model using Linear Regression
* Visualize insights using Power BI

🛠️ Tools & Technologies

* Python (Pandas, NumPy, Scikit-learn)
* Power BI
* API (Frankfurter API)

📊 Features

* Data extraction using API
* Data preprocessing and feature engineering
* 150-day future prediction
* Interactive dashboard

📈 Model Performance

* MAE: 0.2716
* RMSE: 0.3478
* MAPE: 0.29%
* Accuracy: 99.70%

🚀 How to Use

1. Run the Python script
2. Generate predictions
3. Open Power BI file to view dashboard

📌 Conclusion

The model provides highly accurate predictions and can be used for financial analysis and decision-making.
